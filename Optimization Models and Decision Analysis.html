Optimization Models and Decision Analysis

Solving real-world problems using decision analysis and optimization models.

Project Overview

This project explores the use of optimization models and decision analysis techniques to solve practical problems. By leveraging Excel's Solver tool and decision frameworks, this project provides optimized solutions to scenarios involving allocation, cost minimization, and profit maximization.

Key Outcomes

  • Minimized travel time for ambulance dispatch in a city, ensuring timely medical assistance.
  • Optimized cash withdrawals to minimize overall costs over a seven-day period.
  • Maximized profits in production while addressing labor and resource constraints effectively.

Tools Used

  • Microsoft Excel
  • Solver Add-In
  • Optimization Techniques
